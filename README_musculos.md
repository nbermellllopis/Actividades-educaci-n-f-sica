# Actividad: Coloca los músculos

## Objetivo
Actividad interactiva para que el alumnado identifique y relacione los nombres de los músculos con su posición en el cuerpo humano mediante arrastrar y soltar.

## Instrucciones de uso
1. Abrir `musculos.html` en un navegador moderno.
2. Arrastrar cada etiqueta (músculo) y soltarla sobre la zona correspondiente en la imagen.
3. Cuando la colocación sea correcta, la etiqueta se oculta y la zona queda marcada.
4. Usar el botón "Volver a empezar" para reiniciar la actividad.

## Nivel educativo
Dirigido a alumnado de primaria y primer ciclo de ESO.

## Requisitos y notas
- Mantén la carpeta `images/` junto a `musculos.html` para que las imágenes carguen correctamente.
- Evita nombres de archivo con espacios o caracteres especiales (usa `-` o `_`).

## Personalización
- Para cambiar posiciones de las zonas, editar las líneas `div.zona` en `musculos.html` (atributos `left` y `top` en el `style`).
- Para cambiar textos de los músculos, editar los elementos con `class="musculo"` en `musculos.html`.
