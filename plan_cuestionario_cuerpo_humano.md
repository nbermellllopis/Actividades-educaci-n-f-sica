# Plan para crear un cuestionario HTML interactivo sobre el cuerpo humano

## 1. Definir el objetivo y el público
- Enfocar el aprendizaje en los músculos y huesos del cuerpo humano.
- Adaptar el contenido a segundo ciclo de primaria con palabras sencillas, ejemplos visuales y preguntas fáciles de entender.
- Proponer que el alumnado reconozca algunas partes del cuerpo y entienda que los huesos dan sostén y los músculos ayudan a movernos.

## 2. Estructurar el contenido del cuestionario
- Crear 10 preguntas sobre músculos y huesos del cuerpo humano.
- Cada pregunta deberá tener 3 opciones de respuesta: a), b) y c), con una sola respuesta correcta.
- Incluir preguntas sencillas, claras y adecuadas para segundo ciclo de primaria.
- Añadir imágenes del cuerpo humano para hacerlo más visual y atractivo.

## 3. Diseñar la interfaz en HTML
- Crear una página sencilla con título, instrucciones y un área para mostrar las preguntas.
- Usar colores claros, letras grandes y botones fáciles de pulsar.
- Organizar el contenido de forma ordenada y divertida.

## 4. Añadir interactividad
- Hacer que al elegir una respuesta se muestre si es correcta o incorrecta.
- Incluir un botón para pasar a la siguiente pregunta.
- Mostrar un resultado final al terminar el cuestionario.

## 5. Probar y mejorar
- Revisar que todas las preguntas funcionen correctamente.
- Comprobar que se vea bien en pantalla y sea fácil de usar.
- Ajustar el diseño o el texto si hace falta para que sea más claro y motivador.
